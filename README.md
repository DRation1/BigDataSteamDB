# Project_BigDataSteamDB


## 🎮 스팀(Steam) 게임 흥행의 법칙: 가격과 장르가 성공을 결정할까?

### 1. 주제 선정 (Topic Selection)

전 세계 최대 PC 게임 플랫폼인 '스팀(Steam)'에는 매년 수만 개의 게임이 출시됩니다. 하지만 모든 게임이 빛을 보는 것은 아닙니다. 어떤 게임은 '압도적으로 긍정적'인 평가와 함께 대박을 터뜨리지만, 어떤 게임은 소리 소문 없이 사라집니다.

**"과연 비싼 게임이 더 재미있을까? 특정 장르가 성공에 더 유리할까?"**

본 프로젝트는 약 2.7만 개의 스팀 게임 메타데이터를 분석하여, 막연한 통념(비싼 게임 = 갓겜)을 데이터로 검증하고 게임의 상업적 성공(리뷰 수)과 작품성(평점)을 결정짓는 핵심 요인을 탐구하고자 합니다.

### 2. 데이터 수집 (Data Collection)

* **데이터 출처:** Kaggle 'Steam Store Games (Cleaned)' Dataset
* **데이터 규모:** 약 27,075개의 게임 데이터
* **주요 변수:**
    * `Name` (게임명)
    * `Price` (가격)
    * `Genres` (장르)
    * `Positive/Negative Ratings` (긍정/부정 평가 수)
    * `Playtime` (평균 플레이 시간)

### 3. 분석 가설 및 예상 (Hypotheses)

데이터 분석을 시작하기 전, 다음과 같은 가설을 설정하고 검증을 진행했습니다.

1.  **가격 가설:** 게임 가격이 비쌀수록 개발비가 많이 투입되어 평점(만족도)이 높을 것이다.
2.  **장르 가설:** 특정 인기 장르(Action, RPG)가 타 장르보다 높은 평가를 받을 것이다.
3.  **인기도 가설:** 리뷰 수가 많은(유명한) 게임은 평점도 자연스럽게 높을 것이다.
4.  **성공 요인:** 상업적 성공(리뷰 수)과 작품성(평점)을 모두 잡은 '갓겜'들의 공통된 패턴이 존재할 것이다.

### 4. 데이터 전처리 (Data Preprocessing)

원활한 분석을 위해 Pandas를 활용하여 다음과 같은 전처리 과정을 거쳤습니다.

* **파생 변수 생성:**
    * `Total Reviews`: 인기도 척도 (긍정 리뷰 + 부정 리뷰)
    * `Rating Score`: 100점 만점 환산 (긍정 리뷰 비율)
* **Multi-label 문제 해결:**
    * 하나의 게임에 여러 장르(`Action;Indie`)가 섞여 있는 데이터를 `Explode` 기법을 사용해 개별 장르 행으로 분리.
* **이상치 및 결측치 처리:**
    * 분석에 방해되는 노이즈 데이터 제거 및 결측값(0) 보정.

👉 **[데이터 전처리 및 분석 코드 보러가기](./BigDataSteamDB.ipynb)**

### 5. 분석 및 시각화 (Analysis & Visualization)

4가지 관점에서 심층 분석을 수행했습니다.

#### 5-1. 장르별 평점 분포 (Boxplot)
* **분석:** 상위 10개 장르의 평점 분포 비교.
* **결과:** RPG와 Indie 장르의 중앙값이 높게 형성됨. 하위권에는 '실패작(Outlier)'이 다수 존재하지만 상위권은 평준화됨.

#### 5-2. 상관관계 분석 (Heatmap)
* **분석:** 가격, 리뷰 수, 평점 간의 피어슨 상관계수 확인.
* **결과:** 가격과 평점의 상관계수는 `0.08`로 매우 낮음. "비싼 게임이 무조건 재밌지는 않다"는 사실 증명.

#### 5-3. 가격 정책과 품질 (Scatter Plot)
* **분석:** 가격대별 평점 분포 확인.
* **결과:** 저가($20 미만) 시장은 복불복이 심하지만, 고가($60) 시장은 평점 0~40점대의 실패작이 거의 없음. (가격 = 품질 안전망 역할)

#### 5-4. 흥행 성공 매트릭스 (Bubble Chart)
* **분석:** X축(평점) vs Y축(리뷰 수) 4분면 분석.
* **결과:** **Action** 장르는 'High Risk High Return', **Indie** 장르는 작품성은 뛰어나나 노출이 부족한 'Hidden Gems' 구간에 다수 분포.

### 6. 결론 및 인사이트 (Conclusion)

데이터 분석을 통해 얻은 최종 결론은 다음과 같습니다.

1.  **재미의 가성비:** 비싼 게임이 재미를 보장하진 않지만, **실패 확률(똥겜이 될 확률)을 획기적으로 낮춰준다.**
2.  **스팀의 생태계:** 스팀 유저들의 평가는 대체로 관대하며(평점 인플레이션), 경쟁력은 '최악의 평가를 피하는 것'에서 시작된다.
3.  **인디 게임의 생존 전략:** 인디 게임은 품질(평점) 문제는 없으나 발견(리뷰 수)의 문제가 심각하다. 개발만큼이나 '초기 노출 확보(Discoverability)'가 생존의 핵심이다.
